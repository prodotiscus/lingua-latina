Substantivorum Latinorum Declinator 

Программа для склонения латинских существительных Programm für Deklination der lateinischen Substantiven Program for latin nouns declension Programma pro declinatione substantivorum latinorum




    ВНИМАНИЕ!

  • Программа использует для своей работы данные в таком виде, как они обычно указываются в словарях. Например, ursus, i m.
  • Если вы не хотите вводить дополнительные данные о слове, выберите опцию "Использовать данные из словаря" и введите только слово: ursus. Скорее всего, слово есть в словаре.
  • Если в словаре программы слово отсутствует, нужно будет ввести дополнительную словарную информацию (смотрите первый пункт). Отключите опцию "Использовать данные из словаря" и нажмите на кнопку.
    •
    ACHTUNG!

  • Das Programm setzt bei seiner Arbeit Wortdaten so ein, wie sie gewöhnlich in Wörterbüchern vorgestellt sind, z. B.: ursus, i m.
  • Wenn Sie zusätzliche Informationen nach dem Wort nicht eingeben wollen, wählen Sie die Option "Aus dem Wörterbuch Daten einsetzen" und geben Sie nur das Wort ein: ursus. Wahrscheinlich gibt es Informationen nach Ihrem Wort.
  • Wenn es aber nach dem Wort keine Informationen gibt, sollen Sie Zusatzinformationen allerdings eingeben (s. ersten Punkt). Schalten Sie die Option aus und drücken Sie den Button.
    •
    ATTENTION!

  • The program uses in its work word data, as they are usually specified in dictionaries, for example: ursus, i m.
  • If you don't want to input additional information about your word, choose the "Use data from the dictionary" option and input only the word: ursus. Probably program dictionary contains data about the word.
  • However, if the program dictionary doesn't contain it, you should input the additional information (see the first point). Disable the option and click the button.
    •
    ATTENTIO!

  • Programma pro operatione sua data de verbo in visu vocabulario utitur, veluti: ursus, i m.
  • Si informationem additionalem de verbo inducere non vis, optionem "Utere ex vocabulario notitas" elige et tantum verbum induce: ursus. Credibiliter quaeret programma informationem necessariam.
  • Sed si programma nullas notitas quaeret, informationem additionalem inducere debebis (specta punctum primum). Cancella optionem et "Declinare" pressa.
